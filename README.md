# 🛡️ LetsDefend Incident Reports

Welcome to my LetsDefend Incident Reports repository.

This repository documents my hands-on Security Operations Center (SOC) investigations completed on the LetsDefend platform. Each case includes Indicators of Compromise (IOCs), MITRE ATT&CK mappings, timelines, analysis, conclusions, recommendations, and lessons learned.

The goal of this repository is to strengthen my Blue Team and SOC Analyst skills while building a professional cybersecurity portfolio.

---

## 🎯 Objectives

- Learn the SOC investigation workflow
- Practice alert triage and incident response
- Analyze logs and identify suspicious activity
- Document investigations professionally
- Map attacker techniques to the MITRE ATT&CK framework
